# Cancellation：取消执行

LangGraph4j 的流式执行可以取消。

这对长时间运行的图特别有用。

## 怎么取消

图的 `stream()` 会返回一个异步生成器。

你可以调用它的 `cancel(boolean mayInterruptIfRunning)` 来终止执行。

## 两种取消模式

### `cancel(true)`

立即中断当前线程，偏硬。

### `cancel(false)`

让当前节点做完，再在下一个节点前停下，偏温和。

## 什么时候会用

- 用户手动停止
- 超时取消
- 某个条件不满足
- 资源保护

## 为什么这是图式工作流的重要能力

因为 agent 流程不是一次性函数调用，往往会跑很久。

可取消意味着：

- 用户能退出
- 系统能保护资源
- 子图也能跟着被停止
