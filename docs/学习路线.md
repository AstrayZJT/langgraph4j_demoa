# LangGraph4j 学习路线

这个项目按“先最小可运行，再逐步加能力”的方式推进。

## 路线

1. `v0.1.0` 最小 Spring Boot 工程
2. `v0.2.0` 接入 LangGraph4j 最小图
3. `v0.3.0` 条件分支
4. `v0.4.0` 循环
5. `v0.5.0` checkpoint / 持久化
6. `v0.6.0` 和 LangChain4j 结合
7. `v0.7.0` 完整案例

## 这次的起点

先把 LangGraph4j 的最小闭环跑通：

- 定义 `state`
- 定义 `node`
- 连接 `edge`
- 编译 graph
- 启动时执行一次，观察状态流转
