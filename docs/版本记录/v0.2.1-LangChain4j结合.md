# v0.2.1 LangChain4j 结合

## 目标

把 LangChain4j 放进 LangGraph4j 的节点里，验证“流程编排”和“模型调用”可以分层协作。

## 内容

- `WritingAssistant`：用 LangChain4j 注解定义模型接口
- `ToyWritingChatModel`：本地可运行的演示模型
- `LangChainConfig`：把 LangChain4j 代理注册成 Spring Bean
- `DraftWithLangChainNode`：生成草稿
- `PolishWithLangChainNode`：润色草稿
- `LangChainBridgeService`：定义并执行图

## 验证结果

- 项目可编译
- Spring Boot 可启动
- `draft -> polish -> END` 能顺序执行
- LangChain4j 调用结果能写回 state

## 学到的点

- LangGraph4j 负责流程
- LangChain4j 负责单步能力
- `compile()` 是把图定义变成可执行图
- `stream(...)` 可以把图执行过程打印出来
