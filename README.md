# LangGraph4j Demo

这是一个从零开始的 Spring Boot 学习项目，主题是 LangGraph4j。

## 当前阶段

- `v0.1.0` 项目初始化
- `v0.2.0` 最小图：`state` / `node` / `edge`
- `v0.2.1` LangChain4j 结合

## 学习进度

- [LangGraph4j 学习进度清单](./docs/学习进度/README.md)

## 教材文档

- [LangGraph4j 教材目录](./docs/LangGraph4j教材/README.md)

## 运行

```bash
mvn spring-boot:run
```

启动后会在控制台打印每一节 demo 的执行过程。

## 备注

这个项目是学习型工程，重点不是做一个大而全的产品，而是把每个概念都拆开看懂，再一步一步拼起来。
